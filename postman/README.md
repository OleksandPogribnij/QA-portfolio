# Postman Collections

This folder contains Postman collections used for manual API testing and demonstration of QA skills.  

## Contents

- **...** – CRUD operations for Petstore API, including positive scenarios.  
- **Pre-request scripts** – examples of dynamically generated request data.  
- **Tests** – validation scripts checking response status, body fields, and data consistency.  

## Usage

1. Import the `.json` files into Postman (Or copy the RAW link and paste it directly into Postman).  
2. Run individual requests or the full collection to test API endpoints.
